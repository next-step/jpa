## Step2 - 연관 관계 매핑
### 요구사항
- 지하철역과 노선이 있다 
- 지하철역은 여러 노선에 소속될 수 있다
    - 환승역을 고려한다
- 노선 조회 시 속한 지하역을 볼 수 있다
- 지하철역 조회 시 어느 노선에 속한지 볼 수 있다 
    - 지하철역과 노선은 다대다 -> 따라서 StationLine을 생성하여 1:N N:1로 만듬
- 사용자와 즐겨찾기가 있다 (Member, Favorite)
- 사용자는 여러 즐겨찾기를 가진다 (사용자와 즐겨찾기는 1:N)
- 즐겨찾기에는 출발역과 도착역이 있다
